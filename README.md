## ğŸ§© How It Works

1. User opens the SmartSafe Campus web app
2. Location is detected (or entered manually)
3. User saves an emergency contact
4. On pressing **SOS**:
   - Location is pinned on the campus map
   - Alert is logged in the admin dashboard
   - Emergency contact receives a message with a live location link
